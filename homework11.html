You are to design the following model classes. Assume all fields are required unless explicitly identified as optional or unless a default value is provided

The Song model should have the following fields:

  • title — trim whitespace, maximum of 100 characters
  • length — must be string of the form MM:SS where MM is the 2-digit number of minutes and SS is the 2-digit number of seconds

The Album model should have the following fields:

  • title — trim whitespace, maximum of 100 characters
  • artist — trim whitespace, maximum of 100 characters; make an index on this field so we can search by artist
  • date — must be a date; must be between Jan 1, 1900 and current date; default to current date
  • genre — must be one of the following: Pop, Rock, Jazz, Blues, Country, Classical, Reggae, Disco
  • song — an array of references to Song documents

You should turn in two files:

  • song.ts — exports the Song model
  • album.ts — exports the Album model

I should be able to drop your files into my projects and use your models.
